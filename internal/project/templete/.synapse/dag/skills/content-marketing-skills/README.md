# content-marketing-skills

**Content marketing skills — find AI visibility gaps, generate citation-winning content, and push drafts into your CMS.**

## Skills

| Skill | What it does |
|-------|-------------|
| [prompt-gap-to-publish](skills/prompt-gap-to-publish/SKILL.md) | Find weak AI visibility topics and turn them into publish-ready articles, landing pages, or FAQs |
| [citation-recovery-optimizer](skills/citation-recovery-optimizer/SKILL.md) | Improve existing pages to increase AI citations and publish revised drafts to the CMS |
| [competitor-prompt-hijacker](skills/competitor-prompt-hijacker/SKILL.md) | Identify prompts competitors win and generate comparison or rebuttal content to take them back |
| [source-outranker](skills/source-outranker/SKILL.md) | Analyze which third-party sources AI models cite and create stronger source assets designed to outrank them |
| [simulation-backed-publisher](skills/simulation-backed-publisher/SKILL.md) | Generate multiple content variants, evaluate the best one, and publish the winning draft to the CMS |

## Commands

There are currently no commands in this plugin.

## Requirements

These skills are designed to work with Amplitude AI Visibility data via MCP. CMS publishing flows may require additional CMS MCP servers such as WordPress, Webflow, Contentful, Sanity, HubSpot CMS, Ghost, or Shopify if available.
