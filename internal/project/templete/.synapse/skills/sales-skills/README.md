# sales-skills

**Sales skills — pre-call intelligence, account research, and customer briefings for SEs and AEs.**

Each skill is a standalone prompt template designed to help sales teams show up prepared.

## Skills

| Skill | What it does |
|-------|-------------|
| `know-before-you-go-perla-lobera` | Assembles a role-specific pre-call brief from Granola, Salesforce, Outreach, and web signals — so no one walks into a customer call cold. |
